# Brightness

一个极其简陋、纯粹、不建议任何体面人使用的 Windows 外接显示器亮度调节工具。

瞧瞧，伙计。如果你的显示器不支持 DDC/CI 协议，那么它就和隔壁苏珊大妈家那台老掉牙的留声机一样，只能摆着看。当然，现在的显示器大多都挺争气，除非它是从某个上古遗迹里挖出来的宝贝。

它是纯正的 C 语言血统，流淌着原生 UI 的血液。运行内存？哦，上帝，仅仅只有 1.5 MB。这简直比你邻居汤姆口袋里的零钱还要少！

<br>

__听着，伙计，我有言在先__

- 没有自动更新： 我发誓，我真的搞不懂为什么这种小玩意儿也需要更新机制？为了检测那个该死的新版本去浪费哪怕一点点电脑资源，简直是对 CPU 的亵渎，明白吗？

- 界面简直丑得惊动了上帝： 嘿，它只是个拉亮度条的玩意儿。它在那儿，它能用，这就足够了。难道你还想让它给你跳一段百老汇踢踏舞吗？

- 功能简单得像一根火柴： 它现在只会调亮度。如果你想调对比度，听我的，把手伸到显示器屁股后面，去按那些该死的按钮。

## 题外话

说真的，伙计，在 Windows 这个地界上，你就找不到一个哪怕稍微好用那么一点点的亮度工具。

我向上帝保证，我真的会谢。

为什么要装那个笨重的 .NET Core？为什么要用 Electron？为什么要塞进去那些华而不实的圆角、阴影和动画？哦，我的老兄，我只是想偷个懒，不想把手伸到显示器后面去摸索那些该死的按钮而已！
